# excersie-vue
个人学习 Vue 的一些练习

运行项目:
```bash
npm install
npm run dev
```

修改测试 "可复用性&组合/渲染函数&JSX/src/JsxSyntax.jsx" 请运行:
```bash
npm run dev:watchJsx
```

更新 "工具/生产环境部署/src" 后重新编译:
```bash
npm run build:deployment
```

编译并 watch 单文件组件:
```bash
npm run build.w:sfc
```

运行 工具/单元测试
```bash
npm t
```

运行 规模化/路由
```bash
npm run dev.s:baseRounter
```
